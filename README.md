# E-Commerce Store - Capstone Project

## Overview

A fully responsive E-Commerce Web Application built using HTML, CSS, and JavaScript as part of the Web Development Internship Task 5.

## Features

* Product Search
* Category Filtering
* Price Sorting
* Shopping Cart
* Remove Items from Cart
* Total Amount Calculation
* Dark Mode
* Local Storage
* Checkout Form
* Order Confirmation
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Performance Optimization

* Lazy Loading Images
* Responsive Layout
* Optimized UI Components
* Local Storage for Data Persistence
